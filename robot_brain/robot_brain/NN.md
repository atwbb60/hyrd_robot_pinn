### 2. 核心网络模块 (Main Backbone)

#### 第一阶段：局部专家 (Stage 1: Local Experts)

针对 5 个节点分别并行处理。每一个节点包含：

* **输入映射层**: `Linear(15 → 128)` + `LayerNorm(128)` + `GELU` + `Dropout(0.2)`
* **残差块 x2 (ResidualBlock)**:
* `Linear(128 → 128)` → `LayerNorm` → `GELU` → `Dropout`
* `Linear(128 → 128)` → `LayerNorm`
* **Skip Connection**: 



#### 第二阶段：拓扑交互 (Stage 2: Spatial GRU)

利用双向循环网络处理 5 个节点间的空间拓扑连接。

* **Bi-GRU**: `Input(128)`, `Hidden(128)`, `Layers(2)`, `Bidirectional(True)`
* **输出维度**: `[B, 5, 256]` (双向拼接)
* **正则化**: `Dropout(0.2)`

#### 第三阶段：混合输出头 (Stage 3: Hybrid Heads)

特征拼接：将 Stage 1 输出 (128) + Stage 2 输出 (256) + 雅可比特征 (6) 拼接，得到 **390 维**特征。

| 分支 | 结构 (Layer-by-Layer) | 输出 | 激活函数 |
| --- | --- | --- | --- |
| **残差预测头** | `390→128→64→3` |  | `GELU` (前两层) |
| **置信度门控** | `390→64→3` |  | `Tanh` → `Sigmoid` |

> **注**: 门控分支的 Bias 初始值为 **5.0**，强迫训练初期 （完全信任物理模型）。
